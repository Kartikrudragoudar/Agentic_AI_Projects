**Voice Service (backend/app/services/voice/):**

   - whisper_service.py:
     Speech-to-Text using OpenAI Whisper (local, offline, free):
     - Load model: whisper.load_model("base")  # 74MB, fast
     - transcribe(audio_file_path) -> returns transcribed text + detected language code
     - Supports Indian languages: Hindi, Tamil, Telugu, Bengali,
       Marathi, Gujarati, Kannada, Malayalam, Punjabi, Odia natively

   - translation_service.py:
     Translation using IndicTrans2 (AI4Bharat, local, offline, free):
     - Supports 22 Indian languages
     - translate(text, source_lang, target_lang) -> translated text
     - SUPPORTED_LANGUAGES dict mapping language names to codes:
       Hindi: hin_Deva, Tamil: tam_Taml, Telugu: tel_Telu,
       Bengali: ben_Beng, Marathi: mar_Deva, Gujarati: guj_Gujr,
       Kannada: kan_Knda, Malayalam: mal_Mlym, Punjabi: pan_Guru,
       Odia: ory_Orya, English: eng_Latn
     - Pipeline:
       User audio -> Whisper (transcribe) -> IndicTrans2 (to English)
       -> LangGraph Agent -> IndicTrans2 (to user language) -> Response

**AI Agent (backend/app/services/ai/agent.py):**

BankingSupportAgent class built using LangChain and LangGraph frameworks:

**LLM Integration (LangChain):**
- Use ChatGroq from langchain_groq:
  from langchain_groq import ChatGroq
- Initialize:
  llm = ChatGroq(api_key=settings.GROQ_API_KEY, model=settings.GROQ_MODEL)
- Use LangChain message types:
  from langchain_core.messages import SystemMessage, HumanMessage, AIMessage
- Build prompt using ChatPromptTemplate:
  from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
- Use RunnableWithMessageHistory for multi-turn conversation memory:
  from langchain_core.runnables.history import RunnableWithMessageHistory
- Store chat history using ChatMessageHistory:
  from langchain_community.chat_message_histories import ChatMessageHistory

**Agent Graph (LangGraph):**
- Build a stateful agent graph using LangGraph:
  from langgraph.graph import StateGraph, END
- Define AgentState TypedDict with fields:
  messages, session_id, selected_bank, detected_bank,
  detected_service, escalate, calculator_result, language_code
- Graph Nodes:
  - detect_intent_node: detect bank, service, escalation, automatable service
  - calculator_node: run EMI or FD calculator if needed
  - llm_node: call ChatGroq via LangChain chain with enriched context
  - escalation_node: append RBI Ombudsman advisory to response if escalation flagged
- Graph Edges:
  - START -> detect_intent_node
  - detect_intent_node -> calculator_node (if automatable service detected)
  - detect_intent_node -> llm_node (if no automatable service)
  - calculator_node -> llm_node
  - llm_node -> escalation_node (if escalate is True)
  - llm_node -> END (if escalate is False)
  - escalation_node -> END
- Compile graph:
  graph = workflow.compile()

**BankingSupportAgent class:**
- Initializes the LangGraph StateGraph and LangChain ChatGroq LLM
- chat() method invokes the compiled graph with initial state
- Returns dict: response, bank, service, escalate, calculator_result

**Voice Flow Summary:**
1. User holds mic button in browser
2. Browser MediaRecorder captures audio -> sends as blob to POST /api/v1/voice/transcribe/
3. Whisper base model transcribes audio -> detects language automatically
4. IndicTrans2 translates transcribed text from detected language to English
5. English text goes into LangGraph agent -> generates English response
6. IndicTrans2 translates English response back to user's selected language
7. Translated response displayed in chat bubble
